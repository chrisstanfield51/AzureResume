Azure Devops • Github • Azure Blob Storage • CI/CD • CDN • Javascript • Powershell • Azure Functions • Cosmos DB

This site's code was first uploaded to Github. After that it was then pulled into a pipeline with Azure Devops, setup with Continous Intergration and Continous Delivery, which then released onto an Azure Blob Storage account with a CDN setup. The visitor counter is a Javascript code that calls on an Azure Function API written in Powershell that pulls a record from Cosmos DB. It then increments the value by one and returns the new value.
